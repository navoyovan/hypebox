# Project Principles

## Goals

* cinematic and premium feel
* minimal but expressive
* visually polished
* strong typography
* fast loading
* SEO-friendly
* smooth interactions without overwhelming motion

## Priorities

1. visual quality
2. interaction quality
3. performance
4. maintainability
5. feature count

## Avoid

* cluttered UI
* excessive animations
* random gradients
* inconsistent spacing
* unnecessary backend complexity
* giant JS bundles

## Design Philosophy

* whitespace is part of the design
* motion should guide attention
* transitions should feel intentional
* layouts should feel editorial and cinematic
* asymmetry is preferred over rigid grids
