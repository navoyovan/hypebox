# SEO Rules

## Requirements

* every project has its own route
* every page has metadata
* semantic HTML required
* lighthouse score target above 90

## Metadata

Each page should include:

* title
* description
* open graph image

## Performance

* optimize images
* lazy load heavy media
* avoid unnecessary hydration
* minimize JS usage

## Content

* use real text, not text inside images
* proper heading hierarchy
* descriptive project summaries
* accessible markup

## Routing

Good:

* /projects/project-name

Bad:

* everything on one homepage
